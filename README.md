# Almost Equal Sum


## Usage
```
./almost_equal_sum <input_file>
```

## Input file format
<# chunks> <br>
< key1 > < value1 > <br>
< key2 > < value2 > <br>
. <br>
. <br>
. <br>
< keyN > < valueN > <br>

## Example
See "input.txt" for example input file and "out.txt" for output file.
